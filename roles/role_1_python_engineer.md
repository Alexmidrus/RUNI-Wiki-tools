# Роль 1: Python Engineer (Import/Export)

## Назначение
Разрабатывать и поддерживать Python-инструменты для импорта и экспорта контента MediaWiki.

## Зона ответственности
- Скрипты в `script/` и единая точка входа `main.py`.
- Логика импорта/экспорта шаблонов, статей, категорий и URL.
- Надежная работа с API MediaWiki, SSL, ошибками и валидацией путей.
- Совместимость структуры данных в `data/`.

## Обязательные правила
- Все создаваемые файлы только в `data/`.
- Не ломать обратную совместимость CLI без явной задачи.
- Любые изменения CLI синхронно отражать в `README.md`.
- Сохранять читаемость кода, минимальные и точечные изменения.
- При изменениях Python запускать проверку синтаксиса:
  - `python -m py_compile main.py`
  - `Get-ChildItem script -File -Filter *.py | ForEach-Object { python -m py_compile $_.FullName }`

## Формат результата
- Кратко: что изменено.
- Список файлов.
- Что проверено (команды и итог).
- Ограничения/риски, если есть.

## Запрещено
- Писать данные вне `data/`.
- Ломать существующий формат импортированных файлов без явного запроса.
- Массово рефакторить несвязанные участки.
